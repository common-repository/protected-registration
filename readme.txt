== Description ==

A powerful Plugin to protect your Wordpress registration page. 
Why?
A lot of Wordpress users use extra fields Plugins and msot of them want to hide registration fields for non-invited users, using this simple plugin only users with invitation will be able to see registration page.

= Features =
* Easy to install and use.
* Works with your theme's files and UI style.
* A complete help guide for your support.
* Protect your registration page using a single password.
* if you are already using (Easy Invitation Codes) Plugin you can use it as invitation codes source.
* Prevent already logged in users from visiting registration page.

== Installation ==

You can download and install Protected Registration using the built in WordPress plugin installer. If you download the plugin manually, below section describes how to install the Plugin and get it working in 5 simple steps.

1. Upload 'protected-registration' folder to the '/wp-content/plugins/' directory
2. Activate the Plugin through the 'Plugins' menu in WordPress
3. You can see a new section named "Protected Registration" appeared in settings tab in sidebar.
4. Place your settings and click save.
5. You are done.
